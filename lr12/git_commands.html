Основні команди Git (Завдання 2)

1. Create a Repository 2. Observe your Repository
Створити новий локальний репозиторій git init [project name] Перелік незафіксованих або змінених файлів git status
Завантажити з існуючого репозиторію git clone my_url Показати зміни, не додані до індексу git diff
Показати зміни, додані до індексу (staged) git diff --cached
3. Working with Branches 4. Make a change
Перелік усіх гілок git branch -av Додати всі змінені файли до індексу (stage) git add .
Перейти на гілку git checkout my_branch Додати один файл до індексу git add [file]
Створити нову гілку і перейти на неї git checkout -b git_new_branch Зафіксувати staged-файли git commit -m "commit message"
Об'єднати гілку 'b' в 'a' (коли на 'a') git merge branch_b Відкотити зміни до останнього коміту (небезпечно) git reset --hard
5. Synchronize 6. Finally!
Завантажити останні зміни з віддаленого репозиторію (без злиття) git fetch Відправити локальні зміни на віддалений сервер git push
Завантажити та злити зміни git pull Отримати допомогу git command --help
